## 작성 규식
- 형식 : yyyy-mm-dd-제목.md
- 확장자는 .md 또는 .markdown
- 공백이 필요하면 -로 바꿉니다.
- 작성한 파일은 _posts 디렉토리에
```markdown
---
title: "게시글 제목"
author: <author_id> 고정(_date/authors.yml > author_id.name)
categories: [ categories1, categories2 ]
tags: [ tage1, tage2, tage3.... ]
math: true
toc: true
mermaid: true
image: "게시글 상단 메인 이미지"
---
```
